# Искусственный интеллект в информационной безопасности

Добро пожаловать на GitHub репозиторий курса "Искусственный интеллект в информационной безопасности".

Здесь можно найти задания для самостоятельной работы:
1. Первое вводное задание предназначено для ознакомления с предобработкой данных.
2. Второе задание - введение в использовании алгоритмов машинного обучения. 
3. Третье задание демонстрирует применение ML в информационной безопасности.

Все примеры и ресурсы представленные в описании заданий нацелены на использование языка Python.
Тем не менее вы можете не ограничивать себя этим языком.
В том случае, если вы выберете другой язык программирования, то вам нужно будет самостоятельно разобраться в библиотеках
для машинного обучения и анализа данных для выбранного вами языка.
 
Библиотеки для своего любимого языка программирования вы можете найти [здесь](https://github.com/josephmisiti/awesome-machine-learning).
